## Code of Conduct

This project adheres to the [Contributor Covenant](https://www.contributor-covenant.org/). By participating, you are expected to uphold this code.

## Getting Started

1.  **Fork the repository** on GitHub.
2.  **Clone the project** to your local machine.
3.  **Setup your environment**: Follow the [Developer Guide](docs/manuals/developer-guide.md).

## Pull Request Process

1.  Ensure you have self-reviewed your code.
2.  Run any local tests (if applicable).
3.  Update documentation for any new features.
4.  Open a Pull Request with a clear title and description.
5.  Link the PR to the issue it solves.

## Coding Standards

-   **Backend**: TypeScript, Express. Use 4 spaces for indentation (check `.editorconfig` or `tsconfig.json`).
-   **Frontend**: React, TypeScript. Components should be functional.
-   **Commit Messages**: Use imperative mood ("Add feature" not "Added feature").
