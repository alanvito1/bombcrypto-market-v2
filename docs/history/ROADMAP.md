# Roadmap

## Current Status (v1.0.0)

- [x] **Core Marketplace**: Buy/Sell Heroes and Houses.
- [x] **Multi-Chain**: Support for BSC and Polygon.
- [x] **Indexing**: Real-time blockchain event subscribers.
- [x] **API**: Optimized filtering and search.
- [x] **Admin**: Basic cache control.

## Future Plans

### Short Term
- [ ] **Notification System**: Webhooks for real-time sale alerts (partially implemented).
- [ ] **Activity Log**: User-facing history of all sales.
- [ ] **Offer System**: Ability to make offers on NFTs not yet listed.

### Long Term
- [ ] **Auction Mechanism**: Support for timed auctions.
- [ ] **Mobile App Integration**: Native mobile support.
- [ ] **Analytics Dashboard**: Advanced price trends and volume charts.
